# multipleEntry

a mutiple entry template

# 目录
```
src
├─assets //资源文件
│  └─img 
└─pages
    ├─index
    └─other //示例页面
        ├─.*html
        ├─.*css
        └─*js

static //静态文件 如font等
```
# 命令
## 运行 

` npm run dev `

## 打包

` npm run build `
